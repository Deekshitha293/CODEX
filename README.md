# VyaparAI (React + TypeScript + TailwindCSS)

Mobile-first SaaS web app for small shop owners with billing POS, inventory management, expiry alerts, reports, and AI assistant UI.

## Setup

```bash
npm install
npm run dev
```

## Routes

`/`, `/language`, `/category`, `/login`, `/otp`, `/dashboard`, `/billing`, `/billing-history`, `/inventory`, `/add-product`, `/expiry-alerts`, `/ai-assistant`, `/reports`

## Security Implementation (frontend demo)

- JWT token creation utility
- bcrypt password hashing/verification
- AES-GCM encryption utility
- TLS note for HTTPS deployment
